## 🚀 O que é o projeto?

Este é o meu portfólio pessoal, desenvolvido em uma arquitetura Single-Page Application (SPA) ultra leve e estática. Ele adota um **Design System Cyberpunk/Glitch**, desenhado para simular a interface de um terminal de dados invadido ou um HUD (Heads-Up Display) de ficção científica militar.

O objetivo principal é centralizar minha trajetória profissional, proficiência técnica e exibir de forma visualmente impactante os sistemas que construí para o mercado (SaaS, prevenção de perdas, sites comerciais e automações).

---

## ✨ Funcionalidades e Design

- 🎨 **Estética Brutalista** — Uso intenso de `clip-path` para cantos chanfrados, substituindo o padrão de cantos arredondados para um visual mais industrial.
- 📺 **Texturas e Imersão** — Overlay global de *scanlines* simulando monitores CRT antigos e grid de fundo em estilo circuito (teia de dados).
- ⚡ **Animações Glitch** — Efeito contínuo de *Chromatic Aberration* (RGB Split) nos títulos utilizando animações CSS `keyframes`.
- 📊 **HUD de Perfil** — Painel lateral inspirado em sistemas operacionais sci-fi, exibindo métricas de nível, nuvem e barras de status de tecnologias principais.
- 💻 **Terminal Interativo (ABOUT.SH)** — Seção detalhada sobre mim estilizada inteiramente como um console de linha de comando Bash.
- 📱 **Responsividade** — Layout dinâmico que se adapta de maneira fluida a dispositivos mobile (uso otimizado de Grid e Flexbox).
